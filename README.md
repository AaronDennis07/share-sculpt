## Setup
### Clone the repository
- https://github.com/AaronDennis07/share-sculpt.git
### Change the directory
- cd client
### Install dependencies
- npm install or yarn add
### Start the development server
- npm run dev

### Access the app
- Navigate to http://localhost:3000
